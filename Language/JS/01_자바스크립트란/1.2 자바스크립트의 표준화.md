## 1996
8월
- 마이크로소프트가 자바스크립트의 파생 버전인 `JScript`를 출시함.
- 이는 자바스크립트와 완벽하게 호환되지 못하였음.
- 이로 인해 브라우저에 따라 웹페이지가 정상적으로 동작하지 않는 `크로스 브라우징 이슈`가 발생함.

11월
- 넷스케이프 커뮤니케이션즈가 `ECMA 인터내셔널`에 자바스크립트 표준화를 요청함.
## 1997
11월
- `ECMA-262` : 자바스크립트 초판(ECMAScript 1) 사양이 완성됨.
## 1999
- `ECMAScript 3(ES3)`가 공개됨.
## 2009
- `ECMAScript 5(ES5)`가 HTML5과 함께 표준 사양으로 출시됨.
## 2015
- `ECMAScript 6(ES6)`가 공개됨.

## 버전별 특징
| 버전                     | 출시 연도 | 주요 특징                                                        |
| ---------------------- | ----- | ------------------------------------------------------------ |
| ES1                    | 1997  | 초판                                                           |
| ES2                    | 1998  | ISO/IEC 16262 국제 표준과 동일한 규격을 적용                              |
| ES3                    | 1999  | 정규 표현식, try ... catch                                        |
| ~~ES4~~                | 중단됨   | 표준화 위원회 내의 의견 충돌과 기술적 방향성에 대한 이견으로 인해 개발 중단                  |
| ES5                    | 2009  | HTML5와 함께 등장<br>JSON, strict mode, 접근자 프로퍼티, 프로퍼티 어트리뷰트 제어 등 |
| ES6 (ECMAScript 2015)  | 2015  | let/const, 클래스, 화살표 함수, 템플릿 리터럴, 디스트럭처링 할당 등                 |
| ES7 (ECMAScript 2016)  | 2016  | 지수(**) 연산자, Array.prototype.includes 등                       |
| ES8 (ECMAScript 2017)  | 2017  | async/await, Object 정적 메서드 등                                 |
| ES9 (ECMAScript 2018)  | 2018  | Promise.prototype.finally 등                                  |
| ES10 (ECMAScript 2019) | 2019  | Object.fromEntries, Array.prototype.flat 등                   |
| ES11 (ECMAScript 2020) | 2020  | String.prototype.matchAll, BigInt 등                          |
출처 : [ECMAScript version history - Wikipedia](https://en.wikipedia.org/wiki/ECMAScript_version_history)
