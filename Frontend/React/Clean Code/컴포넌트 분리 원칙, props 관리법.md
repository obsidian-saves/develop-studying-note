## 1. 의미
> {{개념명}}이란?  
> 간단히 말해 **무엇을 하는 기술/개념**인지 한 문장으로 요약.

예를 들어,  
1. 어떤 문제를 해결하기 위한 기술인지  
2. 어떤 방식으로 동작하는지  
3. 어디에 쓰이는지 등을 간략히 작성

## 2. 핵심 구성요소
{{개념명}}은(는) 일반적으로 다음과 같은 요소로 구성된다.

1. **요소 1** : 설명  
2. **요소 2** : 설명  
3. **요소 3** : 설명  

> 필요하다면 그림, 코드 블록, 표 등을 함께 첨부 가능.
