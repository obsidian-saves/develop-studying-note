## 1. Git
> Git이란?
> 소스 코드 버전을 관리할 수 있게 지원해주는 시스템

예를 들어,
	개발이 최초로 완료된 소스 코드를 '00 버전'이라 할 때, A라는 사람과 B가 동시에 01 버전을 개발하게 되면 개발 과정에서 충돌이 일어날 수 있다. 이때, Git을 사용하면 이러한 문제점을 해결할 수 있다.
