## 왜 사용해야 하는가?
만약 오픈소스 프로젝트에 라이선스를 설정하지 않으면, 다른 사용자는 해당 소프트웨어가 오픈소스여도 자유롭게 사용할 수 없다.

라이선스를 추가하면 다음과 같은 이점이 따른다.

* **저작권 보호** : 프로젝트에 저작권이 저자에게 있음을 명시한다.
* **사용 허가 범위 명확화** : 다른 사용자가 코드를 사용할 때 어떤 제약을 따르는 지 알 수 있다.
* **협업 촉진** : 명확한 규정이 있어, 사용자는 코드 재사용이나 기여를 더 편하게 할 수 있다.
